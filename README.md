# Swiping Hooks

Easily manage swipe gestures with React hooks.

## Features

- TypeScript support
- Lightweight
- Easy integration
- Live examples with Storybook

## Storybook

To test components and hooks live:

```bash
npm run storybook
```

## Testing

```bash
npm run test
```

## Build

```bash
npm run build
```
